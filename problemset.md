# Problem Set

This file contains the problem requirements and specifications for the microservices hackathon project.

## Project Requirements

- Implement long-running download system
- Handle file IDs from 10K to 100M
- Support asynchronous job processing
- Implement proper error tracking and monitoring
- Deploy to cloud infrastructure

## Key Features

- Job queue with BullMQ
- Distributed tracing with Jaeger
- Error tracking with Sentry
- Real-time status updates
- Presigned S3 URLs for downloads
